# EBP — Distributions

Sampling unit: distribution event (one disbursement to one participant on one date).

## Notes
- Mandatory sample on EBP audits — do not skip
- Distinguish distribution types in scoping: hardship, in-service, termination, RMD, QDRO, death — different documentation requirements
- If the plan permits both rollovers and cash distributions, both should be in scope
- Stratify or pull supplemental selections for higher-risk types (hardship, in-service)
